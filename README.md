# Google Cloud Logging (google-cloud-logging)
Google Cloud Logging is a fully-managed service that performs at scale and can ingest application and system log data from thousands of VMs. Allows you to search, monitor, and analyze log data and events from Google Cloud and AWS.

## APIs

### Google Cloud Logging API
Writes log entries and manages your Cloud Logging configuration. The API is organized into the following resources - Entries (write and list log entries), Logs (list and delete logs), Logs-based metrics (create, update, and delete logs-based metrics), Resource descriptors (list monitored resource descriptors), Sinks (create, update, list, and delete sinks), Exclusions (create, update, list, and delete exclusions), CMEK settings (get and update CMEK settings).

**Human URL:** [https://cloud.google.com/logging](https://cloud.google.com/logging)
